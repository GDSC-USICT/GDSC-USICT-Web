## Getting Started

### Clone the repo
   ```sh
   git clone https://github.com/GDSC-USICT/GDSC-USICT-Web.git && cd ./GDSC-USICT-Web
   ```
### Install packages
   ```sh
   npm install / yarn
   ```
### Start development server
  ```sh
  npm start / yarn start
  ```
### Production build
  ```sh
  npm run build / yarn build
  ```
